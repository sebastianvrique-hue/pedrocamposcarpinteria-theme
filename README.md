# Pedro Campos Carpintería — Tema WordPress

Tema personalizado de WordPress para el sitio de Pedro Campos Carpintería.

## Instalación en cPanel (Git Version Control)

1. En cPanel, abre **Git™ Version Control**.
2. Click en **Create**.
3. **Clone URL**: `https://github.com/sebastianvrique-hue/pedrocamposcarpinteria-theme.git`
4. **Repository Path**: `wp-content/themes/pedrocamposcarpinteria` dentro de la carpeta del sitio (ej. `/home2/manyobra/pedrocamposcarpinteria.cl/wp-content/themes/pedrocamposcarpinteria`).
5. Click **Create**.
6. En WordPress, ve a **Appearance → Themes** y activa "Pedro Campos Carpintería".

## Actualizar el sitio

Cada vez que se hagan cambios y se suban (`git push`) a este repositorio:
1. Entra a cPanel → Git Version Control.
2. Selecciona este repositorio y click en **Manage** → **Pull or Deploy** → **Update from Remote**.
